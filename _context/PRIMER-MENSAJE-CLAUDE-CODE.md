Activa skills /ui-ux-pro-max y /claude.

Lee los archivos PROYECTO.md, SKILLS.md y MEMORIA.md que están en la carpeta _context del proyecto.

Ejecuta /init para generar tu archivo de memoria del proyecto basado en lo que leíste.

Luego haz esto en orden:

PASO 1 — Setup del proyecto:
Crea un proyecto Next.js 14 con App Router llamado joart-nails-studio. Instala: tailwindcss postcss autoprefixer gsap @studio-freight/lenis framer-motion split-type three @types/three. Configura Tailwind con la paleta de colores de JoArt del PROYECTO.md. Configura Google Fonts: Playfair Display, Pinyon Script, Poppins. Copia la carpeta _context al proyecto.

PASO 2 — Crear skills con skill-creator:
Usa /skill-creator para crear las 7 skills del SKILLS.md en orden: /joart-design-system, /joart-animations, /joart-three, /joart-services, /joart-glassmorphism, /joart-svg-botanicals, /joart-deploy.

PASO 3 — GitHub:
Usa el MCP de GitHub para crear el repositorio joart-nails-studio en mi cuenta. Inicializa git en el proyecto, agrega el remote origin y haz el primer commit y push a main. El repo debe ser público.

PASO 4 — Confirma:
Dime la URL del repositorio de GitHub para que yo lo conecte a Vercel. Una vez que yo te confirme que Vercel está conectado, cada cambio que hagas debe terminar con git add . → git commit con mensaje descriptivo en español → git push origin main.

No escribas ningún componente todavía. Solo setup, skills y GitHub.
